Course Description

Covers the fundamentals of full-stack web development and deployment with a strong emphasis on server-side code and functionality. Students will develop a full-stack web application without the use of a pre-existing web server or web framework. Topics include HTTP, APIs, AJAX, databases, encryption, authentication, sockets, privacy, and security.

Lecture Activities / Homework

Most lectures will include an activity that you will work on during lecture. This will be done in a lab-like environment where you can ask the instructor questions and receive guidance on the activity. If you do not complete the activity before the end of lecture you can finish it as homework and it will be due at the beginning of the next lecture.

All activities will be equally weighted.

Team Project

You will also form teams and develop a web application of your choosing. Projects will be graded through a live demonstration on the last day of class.
